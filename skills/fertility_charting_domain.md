# Fertility Charting Domain Knowledge

This application digitizes standardized fertility observation charting.

The system relies exclusively on observable signs.

The system does NOT attempt to predict ovulation.

No probabilistic modeling is allowed.

No machine learning is allowed.

All fertility interpretation is derived from observed data.

The system follows deterministic rules based on mucus observations.

The rules engine determines:

- mucus ranking
- fertile window start
- peak identification
- fertile window end
- cycle phase labels

Terminology must remain neutral and avoid proprietary branding.

The app must not claim affiliation with any specific fertility method organizations.

Allowed terminology:

Observation
Sensation
Appearance
Peak day
Post-peak day
Fertile window

Disallowed terminology:

Official Creighton
Certified Creighton
FertilityCare System

The system must remain method-compatible but brand-neutral.
